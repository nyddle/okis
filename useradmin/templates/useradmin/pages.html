У каждого сайта должна быть минимум одна главная страница, она должна называться index.html. Страницы можно называть только латинскими буквами. Например, myPage.html. В названии страницы сайта не должны присутствовать пробелы, используйте вместо них символ «_». Для создания постраничной навигации, создавайте страницы по образцу: PAGE.pgN.html, где PAGE - название страницы, N - ее номер. Например, book.pg1.html, book.pg2.html и т. д.
